# Intercessor Example

An example app for [Intercessor][intercessor].

## Usage

Install the dependencies:

    npm install

Build it with `intercessor` in `./build`:

    npm run build

Start it:

    npm run start

See it at [localhost:3000](http://localhost:3000).
